Description:
This node is the main, it implements the two behavior of the robot, in the first behavior the robot goes around the room to find new hints, in the second behavior it goes to the oracle with its hypothesis and check if it is the right one. If it's right it tells to us the final sentence instead if is wrong it start again with the first behavior.

def robot.main ( )

This function implements the ros node.

Parameters
None

This function implements the two behaviors of the robot, in the first part it initialises the publisher for the topic /reach and the subscriber for the topic /hypothesis. Then it asks for the current state to the parameters and if state is 1 starts with the first behaviour. Set the position for the room then asks to the service /Change_room if the robot is in the nwe room when it happen it publish on the topic /reach. When the robot have a complete and consistent hypothesis, it goes to the oracle again set the position in the same way as before and send a request to the service /Check_id to ask if the hypothesis is the right one.
