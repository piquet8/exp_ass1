\doxysubsection{Detailed Description}
Documentation for the robot module.

More details.

This node is the main, it implements the two behavior of the robot, in the first behavior the robot goes around the room to find new hints, in the second behavior it goes to the oracle with its hypothesis and check if it is the right one. If it\textquotesingle{}s right it tells to us the final sentence instead if is wrong it start again with the first behavior.
