This function implements the two behaviors of the robot, in the first part it initialises the publisher for the topic /reach and the subscriber for the topic /hypothesis. Then it asks for the current state to the parameters and if state is 1 starts with the first behaviour. Set the position for the room then asks to the service /\+Change\+\_\+room if the robot is in the nwe room when it happen it publish on the topic /reach. When the robot have a complete and consistent hypothesis, it goes to the oracle again set the position in the same way as before and send a request to the service /\+Check\+\_\+id to ask if the hypothesis is the right one.
